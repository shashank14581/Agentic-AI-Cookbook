Pattern 5 · JSON

Agents Returning Structured JSON

In this pattern, agents stop returning only free-form text. They return structured JSON that can be parsed, validated, routed, logged, and used by other software systems.

The agent becomes machine-readable.

JSON makes the model output usable by code, not just humans.

OVERVIEW

What this pattern teaches

A StructuredAgent is introduced with a system prompt that enforces a JSON schema. The agent acts like a conversation analyst: given any message, it returns a JSON object specifying speaker, mood, topic, sentiment, and reply.

A helper method parses the raw response into Python data while handling cases where the model wraps JSON in markdown backticks. The orchestrator demonstrates how human, dog, and analyst agents can work together in one loop.

import os
import json
from google import genai
from google.genai import types

# Base Agent
class BaseAgent():

    def __init__(self, name, sys_prompt, model="gemini-2.5-flash"):

        self.name = name
        self.sys_prompt = sys_prompt
        self.memory = []

        self.client = genai.Client(
            api_key=os.environ.get("GEMINI_API_KEY")
        )

        self.model = model

    def think(self, input_text):

        contents = [
            types.Content(
                role="user",
                parts=[types.Part.from_text(text=input_text)]
            )
        ]

        config = types.GenerateContentConfig(
            thinking_config=types.ThinkingConfig(thinking_budget=0),
            system_instruction=[types.Part.from_text(text=self.sys_prompt)]
        )

        response = ""

        for chunk in self.client.models.generate_content_stream(
            model=self.model,
            contents=contents,
            config=config
        ):

            if chunk.text:
                response += chunk.text

        self.memory.append((input_text, response))

        return response

# Structured Agent
class StructuredAgent(BaseAgent):

    def __init__(self, name):

        super().__init__(
            name,
            """You are a conversation analyst.

Given a message, return ONLY a JSON object.
No explanation, no markdown, no backticks.

Schema:
{
  "speaker": "who is speaking, Human or Dog",
  "mood": "one word mood of the speaker",
  "topic": "one sentence about what is being discussed",
  "sentiment": "positive | negative | neutral",
  "reply": "a natural reply from the other party"
}

Return only valid JSON. Nothing else."""
        )

    def think_structured(self, input_text):

        raw = self.think(input_text)

        clean = raw.strip()

        if clean.startswith("```"):

            clean = clean.split("```")[1]

            if clean.startswith("json"):
                clean = clean[4:]

        clean = clean.strip()

        try:

            parsed = json.loads(clean)

            return parsed

        except json.JSONDecodeError:

            return {
                "error": "Failed to parse JSON",
                "raw": raw
            }

# Human Agent
class Human(BaseAgent):

    def __init__(self, name):

        super().__init__(
            name,
            f"You are {name}, you are a human. Talk naturally."
        )

# Dog Agent
class Dog(BaseAgent):

    def __init__(self, name):

        super().__init__(
            name,
            f"You are {name}, you are a dog. Bark, be playful."
        )

# Orchestrate
def orchestrate(human, dog, analyst, turns=3):

    message = "Hi Dog!"

    for i in range(turns):

        print(f"\\n{'='*50}")
        print(f"Turn {i+1}")
        print(f"{'='*50}")

        print(f"\\n{human.name}: ", end="")

        human_out = human.think(message)

        print(human_out)

        print(f"\\n{dog.name}: ", end="")

        dog_out = dog.think(human_out)

        print(dog_out)

        print(f"\\n{analyst.name} analysis:")

        structured = analyst.think_structured(dog_out)

        print(json.dumps(structured, indent=2))

        message = dog_out

# Main
if __name__ == "__main__":

    human = Human("Shashank")
    dog = Dog("Bruno")
    analyst = StructuredAgent("Analyst")

    orchestrate(human, dog, analyst, turns=2)
Mental Model

Lines 1–4: Import os, json, and the Gemini SDK. The json module is used to parse structured model output.

Lines 7–46: Define the basic BaseAgent. It sends input to Gemini, streams the response, stores the interaction in memory, and returns raw text.

Lines 49–77: Define StructuredAgent. Its system prompt tells Gemini to return only valid JSON following a strict schema.

Lines 79–106: The think_structured() method cleans the raw model output, removes markdown fences if present, and attempts to parse the result using json.loads().

Lines 109–124: Define simple Human and Dog agents using natural role-based prompts.

Lines 127–154: The orchestrator runs a conversation turn, lets the dog reply, and then asks the analyst to convert the dog’s reply into structured JSON.

Lines 157–162: Create the human, dog, and analyst agents. The final run shows how structured outputs can sit beside normal conversational agents.
