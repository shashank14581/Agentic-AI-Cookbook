Pattern 5: Returning Structured JSON

This pattern teaches an agent to output structured JSON instead of free‑form text.

Summary

A StructuredAgent is introduced with a system prompt that enforces a JSON schema. The agent acts like a conversation analyst: given any message, it returns a JSON object specifying the speaker, mood, topic, sentiment and a reply. A helper method parses the raw response into Python data while gracefully handling cases where the model wraps the JSON in backticks. The orchestrator demonstrates how to involve three agents (human, dog, analyst) in a single loop.


import os
import json
from google import genai
from google.genai import types

# -------- BASE AGENT --------
class BaseAgent():
    def __init__(self, name, sys_prompt, model="gemini-2.5-flash"):
        self.name = name
        self.sys_prompt = sys_prompt
        self.memory = []
        self.client = genai.Client(api_key=os.environ.get("GEMINI_API_KEY"))
        self.model = model
    def think(self, input_text):
        contents = [
            types.Content(
                role="user",
                parts=[types.Part.from_text(text=input_text)]
            )
        ]
        config = types.GenerateContentConfig(
            thinking_config=types.ThinkingConfig(thinking_budget=0),
            system_instruction=[types.Part.from_text(text = self.sys_prompt)]
        )
        response = ""
        for chunk in self.client.models.generate_content_stream(
            model=self.model,
            contents=contents,
            config=config
        ):
            if chunk.text:
                response += chunk.text
        self.memory.append((input_text, response))
        return response

# -------- STRUCTURED AGENT --------
class StructuredAgent(BaseAgent):
    def __init__(self, name):
        super().__init__(
            name,
            """You are a conversation analyst.

Given a message, return ONLY a JSON object. No explanation, no markdown, no backticks.

Schema:
{
  "speaker": "who is speaking (Human or Dog)",
  "mood": "one word mood of the speaker",
  "topic": "one sentence — what is being talked about",
  "sentiment": "positive | negative | neutral",
  "reply": "a natural reply from the other party"
}

Return only valid JSON. Nothing else."""
        )
    def think_structured(self, input_text):
        raw = self.think(input_text)
        clean = raw.strip()
        if clean.startswith("```"):
            clean = clean.split("```")[1]
            if clean.startswith("json"):
                clean = clean[4:]
        clean = clean.strip()
        try:
            parsed = json.loads(clean)
            return parsed
        except json.JSONDecodeError:
            return {"error": "Failed to parse JSON", "raw": raw}

# -------- HUMAN AGENT --------
class Human(BaseAgent):
    def __init__(self, name):
        super().__init>(
            name,
            f"You are {name}, you are a human. Talk naturally."
        )

# -------- DOG AGENT --------
class Dog(BaseAgent):
    def __init__(self, name):
        super().__init>(
            name,
            f"You are {name}, you are a dog. Bark, be playful."
        )

# -------- ORCHESTRATE --------
def orchestrate(human, dog, analyst, turns=3):
    message = "Hi Dog!"
    for i in range(turns):
        print(f"\n{'='*50}")
        print(f"Turn {i+1}")
        print(f"{'='*50}")
        print(f"\n{human.name}: ", end="")
        human_out = human.think(message)
        print(human_out)
        print(f"\n{dog.name}: ", end="")
        dog_out = dog.think(human_out)
        print(dog_out)
        print(f"\n{analyst.name} analysis:")
        structured = analyst.think_structured(dog_out)
        print(json.dumps(structured, indent=2))
        message = dog_out

# -------- MAIN --------
if __name__ == "__main__":
    human = Human("Shashank")
    dog = Dog("Bruno")
    analyst = StructuredAgent("Analyst")
    orchestrate(human, dog, analyst, turns=2)

Lines 1–3: Import os, json and the Gemini client. This pattern uses json to parse structured outputs.

Lines 10–38: Define the basic agent class, identical to earlier patterns. The think() method streams the model’s output.

Lines 41–63: Define the StructuredAgent. Its system prompt specifies a JSON schema that the model must obey. The think_structured() helper calls think(), strips any markdown fences, and attempts to parse the returned text as JSON. If parsing fails, it returns an error structure.

Lines 86–103: Define human and dog agents using simple prompts, just like in earlier patterns.

Lines 106–133: Define orchestrate(). It runs multiple turns where the human speaks, the dog replies, and then the analyst processes the dog’s reply into a JSON analysis. The json.dumps() call pretty‑prints the analysis dictionary.

Lines 135–142: In the main block, create all three agents and run two turns of analysis. The output demonstrates that the analyst produces structured JSON for each dog response.
