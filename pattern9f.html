Pattern 9f – Basic RAG Agent

This agent retrieves documents from a small vector store before answering. It then asks Gemini to respond using only the retrieved context.

10 class SimpleEmbeddingModel(nn.Module):
11     def __init__(self, vocab_size, embedding_dim=32):
12       super().__init__()
13       self.embedding = nn.Embedding(vocab_size, embedding_dim)
14   
15     def forward(self, token_ids):
16       word_embeddings = self.embedding(token_ids)
17       return word_embeddings.mean(dim=0)

19 class RAGMemory:
20     def __init__(self, embedding_dim=32):
21       self.word_to_id = {"<UNK>": 0}
22       self.embedding_dim = embedding_dim
23       self.model = None
24       self.store = []
25 
26     def build_vocab(self, texts):
27       for text in texts:
28         for word in text.lower().split():
29           if word not in self.word_to_id:
30             self.word_to_id[word] = len(self.word_to_id)
31 
32     def encode_text(self, text):
33       token_ids = [
34         self.word_to_id.get(word, 0)
35         for word in text.lower().split()
36       ]
37       return torch.tensor(token_ids, dtype=torch.long)
38 
39     def build_index(self, knowledge_base):
40       self.build_vocab(knowledge_base)
41       self.model = SimpleEmbeddingModel(
42           vocab_size = len(self.word_to_id),
43           embedding_dim = self.embedding_dim
44       )
45       for i, text in enumerate(knowledge_base, start=1):
46         token_ids = self.encode_text(text)
47         with torch.no_grad():
48           embedding = self.model(token_ids)
49         self.store.append({
50             "doc_id": f"doc_{i}",
51             "text": text,
52             "embedding": embedding
53         })
54         print(f"Indexed doc_{i}")
55 
56     def retrieve(self, query, top_k=3):
57       query_ids = self.encode_text(query)
58       with torch.no_grad():
59         query_embedding = self.model(query_ids)
60       results = []
61       for item in self.store:
62         score = F.cosine_similarity(
63             query_embedding.unsqueeze(0),
64             item["embedding"].unsqueeze(0)
65         ).item()
66         results.append((score, item))
67       results = sorted(results, key=lambda x: x[0], reverse=True)
68       context = ""
69       for score, item in results[:top_k]:
70         context += f"""
71   Document: {item["doc_id"]}
72   Similarity: {round(score, 4)}
73   Text: {item["text"]}
74   """
75       return context

Embedding & Memory

The SimpleEmbeddingModel takes a vocabulary size and outputs an average embedding for any list of token IDs. The RAG memory builds a vocabulary from the knowledge base, constructs the embedding model, and stores each document with its vector representation.

The build_index() method tokenises each document, computes an embedding using SimpleEmbeddingModel, and appends a record containing the document ID, text and embedding to the store.

Retrieval

When a query arrives, RAGMemory.retrieve() encodes the query, computes its embedding and calculates cosine similarity against each stored document. The top k results are formatted into a human‑readable context that includes document identifiers, similarity scores and the original text.

Agent Thinking

The BaseAgent receives a question, calls retrieve() on the RAG memory and builds a final prompt instructing Gemini to answer using only the retrieved context. It prints and collects the model’s streamed response.

The derived BasicRAGAgent simply sets a helpful system prompt: it reminds the model that it is a RAG agent and must not hallucinate outside the retrieved data.

Running the Example

In the __main__ block a knowledge base of project facts is indexed and a BasicRAGAgent is created. The orchestrate function loops through a list of user questions, prints each question and hands control to the agent. The agent will always retrieve context before generating an answer.
