Personality and Memory

OVERVIEW

What this pattern teaches

The BaseAgent gains a facts store and a context builder. Before sending input to Gemini, the agent extracts sentences like I am ... or I like ... and stores them as long-term facts.

When generating a response, the agent constructs a prompt that includes both its facts and the last few interactions. Specialized agents such as a human and a god demonstrate how different system prompts create different personalities.

import os
from google import genai
from google.genai import types

class BaseAgent():

    def __init__(self, name, sys_prompt, model="gemini-2.5-flash-lite"):

        self.name = name
        self.sys_prompt = sys_prompt
        self.memory = []
        self.facts_store = []

        self.client = genai.Client(
            api_key=os.environ.get("GEMINI_API_KEY")
        )

        self.model = model

    def build_context(self, input_text):

        history = ""

        for inp, out in self.memory[-3:]:

            history += f"user: {inp}\nsystem: {out}\n"

        facts_text = "\n".join(self.facts_store)

        full_input = f"""
FACTS:
{facts_text}

HISTORY:
{history}

NEW MESSAGE:
{input_text}
"""

        return full_input

    def extract_facts(self, input_text):

        if "I am" in input_text or "I like" in input_text:

            if input_text not in self.facts_store:

                self.facts_store.append(input_text)

    def think(self, input_text):

        self.extract_facts(input_text)

        full_input = self.build_context(input_text)

        contents = [

            types.Content(
                role="user",
                parts=[types.Part.from_text(text=full_input)]
            )

        ]

        config = types.GenerateContentConfig(

            thinking_config=types.ThinkingConfig(
                thinking_budget=0
            ),

            system_instruction=[
                types.Part.from_text(text=self.sys_prompt)
            ]

        )

        response = ""

        for chunk in self.client.models.generate_content_stream(

            model=self.model,
            contents=contents,
            config=config

        ):

            if chunk.text:

                print(chunk.text, end="")
                response += chunk.text

        print()

        self.memory.append((input_text, response))

        return response

class Human(BaseAgent):

    def __init__(self, name):

        super().__init__(
            name,
            f"You are {name}, a human. Talk naturally and casually."
        )

class God(BaseAgent):

    def __init__(self, name):

        super().__init__(
            name,
            f"You are {name}, a Hindu God. Be divine, wise, slightly playful."
        )

def orchestrate(human, god, turns=5):

    message = "Hi Indra!"

    for _ in range(turns):

        print(f"\n{human.name}: ", end="")

        human_out = human.think(message)

        print(f"\n{god.name}: ", end="")

        god_out = god.think(human_out)

        message = god_out

if __name__ == "__main__":

    human = Human("Shashank")
    god = God("Indra")

    orchestrate(human, god)
Mental Model

Lines 1–3: Import os and the Gemini SDK modules. These provide access to the Gemini client and type definitions.

Lines 5–18: Initialize the BaseAgent. This pattern adds facts_store, which acts as a simple long-term memory.

Lines 20–40: Define build_context(). It creates a prompt containing facts, recent conversation history, and the new user message.

Lines 42–50: Define extract_facts(). If the input contains phrases such as I am or I like, the agent stores that sentence as a fact.

Lines 52–91: The think() method now performs three jobs: extract facts, build context, and send the enriched prompt to Gemini.

Lines 93–100: Define the Human subclass. Its prompt makes the agent speak naturally and casually.

Lines 102–109: Define the God subclass. Its prompt creates a divine, wise, slightly playful personality.

Lines 111–122: The orchestrator alternates between the human and the god. The output from one agent becomes the input for the other.

Lines 124–128: The script creates a human agent and a god agent, then launches the conversation. Over time, memory and personality shape the dialogue.
