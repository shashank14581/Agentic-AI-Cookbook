Pattern 9l – Agent + MCP: Full Integration

An agent uses Gemini to reason and calls multiple tools via an MCP server. The tools provide private knowledge and compute risk scores, enabling complex decision making.

56 # MCP CLIENT
57 class MCPClient:
58   def __init__(self, url=MCP_URL):
59     self.url = url
60   async def call_tool(self, tool_name, arguments):
61     async with streamablehttp_client(self.url) as (read, write, _):
62       async with ClientSession(read, write) as session:
63         await session.initialize()
64         result = await session.call_tool(name=tool_name,arguments=arguments)
65         return result.content[0].text

85   async def think(self, input_text):
86     response = self.client.models.generate_content(
87       model=self.model,
88       contents=[types.Content(role="user",parts=[types.Part.from_text(text=input_text)])],
89       config=types.GenerateContentConfig(thinking_config=types.ThinkingConfig(thinking_budget=0),system_instruction=[types.Part.from_text(self.sys_prompt)],tools=self.get_tools())
90     )
91     part = response.candidates[0].content.parts[0]
92     if part.function_call:
93       tool_name = part.function_call.name
94       arguments = dict(part.function_call.args)
95       print(f"\n[Gemini selected MCP tool: {tool_name}]")
96       print(arguments)
97       answer = await self.mcp_client.call_tool(tool_name=tool_name,arguments=arguments)
98     else:
99       answer = response.text
100     print(answer)
101     self.memory.append((input_text, answer))
102     return answer

Server with Multiple Tools

The server defines two tools: search_private_knowledge for retrieving internal facts about Blue Mango, Silver Gate or confidence thresholds, and calculate_campaign_risk for mapping a numeric confidence score to a risk assessment. Tools are registered via decorators on functions and served via FastMCP.

Client and Base Agent

The MCPClient and the BaseAgent are extended to handle multiple tools. The get_tools() method returns a list of two function declarations. When Gemini decides to call a tool, the agent prints the selection and passes the call to the MCP client. Responses are printed and stored in memory.

MCP Agent Prompt

The MCPAgent sets a system prompt that instructs Gemini exactly when to use each tool. It clarifies that search_private_knowledge should be used for questions about Blue Mango, Silver Gate or other private facts, and that calculate_campaign_risk should be used when a confidence score is provided. The prompt prohibits fabrication of private information.

Bringing It All Together

In the main() coroutine the server is started if necessary, a client and MCP agent are created, and a series of questions are asked. Some queries trigger retrieval of private knowledge, others request a risk calculation, and the agent seamlessly switches between tool calls and direct replies. This pattern demonstrates full integration between LLM reasoning and an external tool server.
