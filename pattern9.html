The Debate Pattern

In this pattern, multiple agents with conflicting personas argue over a topic. By forcing agents to defend opposing views, we surface nuances, detect biases, and reach a more balanced conclusion.

OVERVIEW

What this pattern teaches

A Debate Pattern uses three distinct agents: a Proponent, an Opponent, and a Judge. The goal isn't just to generate text, but to iterate through several rounds of arguments.

The Proponent presents a case, the Opponent finds flaws, and the cycle repeats. Finally, the Judge reviews the entire transcript to provide a neutral, comprehensive verdict. This helps mitigate "hallucinations" and one-sided logic.

CODE

Line-by-line walkthrough

pattern9.py 
import os
from google import genai

class Debater:
    def __init__(self, name, persona):
        self.name = name
        self.persona = persona
        self.client = genai.Client(api_key=os.getenv("GEMINI_API_KEY"))

    def argue(self, context):
        prompt = f"{self.persona}\n\nContext:\n{context}\n\nYour turn:"
        response = self.client.models.generate_content(
            model="gemini-2.5-flash", contents=prompt
        )
        return response.text

def conduct_debate(topic, rounds=2):
    pro = Debater("Pro", f"Defend the idea: {topic}")
    con = Debater("Con", f"Oppose the idea: {topic}")
    judge = Debater("Judge", "You are a neutral judge. Summarize both sides.")
    
    history = f"Topic: {topic}\n"
    
    for i in range(rounds):
        pro_arg = pro.argue(history)
        history += f"\nPro: {pro_arg}"
        
        con_arg = con.argue(history)
        history += f"\nCon: {con_arg}"
        
    verdict = judge.argue(history)
    return verdict
Mental Model

Lines 4-15: The Debater class encapsulates a specific viewpoint. The argue method takes the current history (the context) to generate the next logical counter-point.

Lines 18-21: We initialize three specialized instances. By assigning them strict, opposing personas, we force Gemini to explore the edge cases of the topic.

Lines 25-31: The main debate loop. Note how the history grows each turn. Each agent "sees" what the other just said, allowing for direct rebuttals rather than isolated statements.

Line 33: The Judge doesn't participate in the fight. They wait until the "dust settles" to analyze the quality of arguments from both sides.
