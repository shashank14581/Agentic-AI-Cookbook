Pattern 9 · Fan-Out / Fan-In

Debate Agent Workflow

This pattern demonstrates a multi-agent debate system. Two debaters speak in parallel, two critics cross-examine in parallel, and a judge performs the final fan-in by combining all arguments into a verdict.

OVERVIEW

What this pattern teaches

Pattern 9 shows how multiple agents can work in parallel and then converge into one final answer. The first fan-out creates two opposing positions at the same time. The second fan-out creates two critiques at the same time. The fan-in stage sends all outputs to a judge agent.

This pattern is useful for adversarial reasoning, decision review, argument testing, critique loops, and any workflow where independent perspectives need to be generated before a final synthesis.

import os
import asyncio
from google import genai
from google.genai import types


# ── Base Agent ────────────────────────────────────────────────────────────────

class BaseAgent:
    def __init__(self, name: str, sys_prompt: str, model: str = "gemini-2.5-flash-lite"):
        self.name = name
        self.sys_prompt = sys_prompt
        self.memory: list[tuple[str, str]] = []
        self.client = genai.Client(api_key=os.environ["GEMINI_API_KEY"])
        self.model = model
        self._config = types.GenerateContentConfig(
            thinking_config=types.ThinkingConfig(thinking_budget=0),
            system_instruction=[types.Part.from_text(text=self.sys_prompt)],
        )

    async def think(self, input_text: str) -> str:
        contents = [
            types.Content(
                role="user",
                parts=[types.Part.from_text(text=input_text)],
            )
        ]
        response = ""
        stream = await self.client.aio.models.generate_content_stream(
            model=self.model,
            contents=contents,
            config=self._config,
        )
        async for chunk in stream:
            if chunk.text:
                print(chunk.text, end="", flush=True)
                response += chunk.text
        print()
        self.memory.append((input_text, response))
        return response


# ── Agent Roles ───────────────────────────────────────────────────────────────

class Debater(BaseAgent):
    def __init__(self, name: str, stance: str):
        super().__init__(
            name,
            f"You are {name}. You firmly believe: {stance}. "
            "State your position in 3 sharp sentences. No hedging.",
        )


class Critic(BaseAgent):
    def __init__(self, name: str):
        super().__init__(
            name,
            f"You are {name}, a sharp cross-examiner. "
            "You will be given an opponent's argument. "
            "Identify its weakest point and attack it in 2 sentences.",
        )


class Judge(BaseAgent):
    def __init__(self):
        super().__init__(
            "Judge",
            "You are an impartial judge. "
            "You will receive two positions and two critiques. "
            "Weigh the arguments. Pick the stronger side. "
            "State your verdict and reasoning in 3 sentences. "
            "Do not sit on the fence.",
        )


# ── Stage 1 — Fan-Out: Parallel Positions ────────────────────────────────────

async def opening_positions(topic: str, debater_a: Debater, debater_b: Debater) -> tuple[str, str]:
    print(f"\n{'='*60}")
    print(f"TOPIC: {topic}")
    print(f"{'='*60}")
    print("\n[Stage 1] Opening positions …\n")

    async def speak(agent: Debater) -> str:
        print(f"\n── {agent.name} ──")
        return await agent.think(f"State your position on: {topic}")

    position_a, position_b = await asyncio.gather(
        speak(debater_a),
        speak(debater_b),
    )

    return position_a, position_b


# ── Stage 2 — Cross-Critique: Each Attacks the Other ─────────────────────────

async def cross_critique(
    debater_a: Debater, position_a: str,
    debater_b: Debater, position_b: str,
) -> tuple[str, str]:
    print("\n[Stage 2] Cross-critique …\n")

    critic_a = Critic(f"{debater_a.name} (cross-examining)")
    critic_b = Critic(f"{debater_b.name} (cross-examining)")

    async def attack(critic: Critic, opponent_name: str, opponent_position: str) -> str:
        print(f"\n── {critic.name} ──")
        return await critic.think(f"{opponent_name} argued: {opponent_position}")

    critique_by_a, critique_by_b = await asyncio.gather(
        attack(critic_a, debater_b.name, position_b),
        attack(critic_b, debater_a.name, position_a),
    )

    return critique_by_a, critique_by_b


# ── Stage 3 — Fan-In: Judge Decides ──────────────────────────────────────────

async def verdict(
    debater_a: Debater, position_a: str, critique_by_b: str,
    debater_b: Debater, position_b: str, critique_by_a: str,
) -> str:
    print("\n[Stage 3] Verdict …\n")

    judge = Judge()

    prompt = (
        f"{debater_a.name} argued:\n{position_a}\n\n"
        f"{debater_b.name} cross-examined {debater_a.name}:\n{critique_by_b}\n\n"
        f"{debater_b.name} argued:\n{position_b}\n\n"
        f"{debater_a.name} cross-examined {debater_b.name}:\n{critique_by_a}\n\n"
        "Render your verdict."
    )

    print("── Judge ──")
    return await judge.think(prompt)


# ── Main ──────────────────────────────────────────────────────────────────────

async def main():
    topic = "AI will eliminate more jobs than it creates in the next decade"

    debater_a = Debater("Arjun", "AI is a net job destroyer — history will not repeat itself this time")
    debater_b = Debater("Meera", "AI is a net job creator — every technology wave has expanded human work")

    position_a, position_b = await opening_positions(topic, debater_a, debater_b)

    critique_by_a, critique_by_b = await cross_critique(
        debater_a, position_a,
        debater_b, position_b,
    )

    final_verdict = await verdict(
        debater_a, position_a, critique_by_b,
        debater_b, position_b, critique_by_a,
    )

    print("\n" + "="*60)
    print("VERDICT")
    print("="*60)
    print(final_verdict)


import nest_asyncio
nest_asyncio.apply()

if __name__ == "__main__":
    asyncio.run(main())
Mental Model

Lines 1-4: Import async support and Gemini SDK components. The whole pattern depends on concurrent agent calls.

Lines 9-40: BaseAgent wraps Gemini streaming, stores memory, and exposes an async think() method that every role can reuse.

Lines 45-75: Three roles are defined: Debater argues from a fixed stance, Critic attacks weak points, and Judge weighs everything and decides.

Lines 82-101: Stage 1 is fan-out. asyncio.gather() lets Arjun and Meera produce opening positions at the same time.

Lines 108-130: Stage 2 is another fan-out. Each side cross-examines the other position concurrently through critic agents.

Lines 137-160: Stage 3 is fan-in. The judge receives both arguments and both critiques in one combined prompt, then renders a verdict.

Lines 166-195: The main function wires the debate together: topic, debaters, opening positions, critiques, verdict, and final printed output.

Lines 198-202: nest_asyncio makes this friendlier in notebook environments, while asyncio.run() runs the workflow as a script.
