Pattern 8: Parallel Agents (Fan‑Out & Fan‑In)

Multiple agents work concurrently, then their outputs are critiqued and synthesized.

Summary

This pattern uses Python’s asyncio to run multiple agents in parallel. In stage 1, several analysts generate independent insights about a topic. In stage 2, a critic evaluates each analyst’s output concurrently. Stage 3 merges the analyses and critiques into a coherent summary. This fan‑out/fan‑in approach illustrates how to scale reasoning by parallelizing tasks and then synthesizing the results.


import os, asyncio, nest_asyncio
from google import genai
from google.genai import types

# Async BaseAgent: uses async think()
class BaseAgent:
    def __init__(self, name, sys_prompt, model="gemini-2.5-flash-lite"):
        self.name = name
        self.sys_prompt = sys_prompt
        self.memory = []
        self.client = genai.Client(api_key=os.environ["GEMINI_API_KEY"])
        self.model = model
        self._config = types.GenerateContentConfig(
            thinking_config=types.ThinkingConfig(thinking_budget=0),
            system_instruction=[types.Part.from_text(text=self.sys_prompt)],
        )
    async def think(self, input_text):
        contents = [types.Content(role="user", parts=[types.Part.from_text(text=input_text)])]
        response = ""
        stream = await self.client.aio.models.generate_content_stream(model=self.model, contents=contents, config=self._config)
        async for chunk in stream:
            if chunk.text:
                print(chunk.text, end="", flush=True)
                response += chunk.text
        print()
        self.memory.append((input_text, response))
        return response

# Roles: Analyst, Critic, Synthesizer
class Analyst(BaseAgent):
    def __init__(self, name, domain):
        prompt = f"You are {name}, a sharp {domain} analyst. Give a concise 2-sentence insight."
        super().__init__(name, prompt)

class Critic(BaseAgent):
    def __init__(self, name):
        super().__init>(name, f"You are {name}, a ruthless critic. Identify the single biggest flaw. One sentence.")

class Synthesizer(BaseAgent):
    def __init__(self, name):
        super().__init>(name, f"You are {name}, a synthesis engine. Merge all inputs into one coherent paragraph.")

# Stage 1: Parallel analysts
async def run_parallel_analysts(topic):
    analysts = [
        Analyst("Riya",  "market economics"),
        Analyst("Arjun", "geopolitical risk"),
        Analyst("Meera", "technology adoption"),
    ]
    async def run(agent):
        print(f"── {agent.name} ──")
        return agent.name, await agent.think(topic)
    results = await asyncio.gather(*[run(a) for a in analysts])
    return dict(results)

# Stage 2: Parallel critics
async def run_parallel_critics(analyst_outputs):
    async def critique(name, text):
        critic = Critic("Dev")
        print(f"── Critic on {name} ──")
        return name, await critic.think(f"{name} said: {text}")
    results = await asyncio.gather(*[critique(n, t) for n, t in analyst_outputs.items()])
    return dict(results)

# Stage 3: Synthesis
async def synthesize(analyst_outputs, critic_outputs):
    synth = Synthesizer("Priya")
    combined = "\n".join(
        f"{name}: {analysis}\n(Critique: {critic_outputs[name]})"
        for name, analysis in analyst_outputs.items()
    )
    return await synth.think(f"Merge these views:\n\n{combined}")

# Main: fan-out then fan-in
async def main():
    topic = "The rise of sovereign AI compute infrastructure in emerging economies"
    analysts = await run_parallel_analysts(topic)
    critiques = await run_parallel_critics(analysts)
    final = await synthesize(analysts, critiques)
    print(final)

nest_asyncio.apply()
if __name__ == "__main__":
    asyncio.run(main())

Asynchronous BaseAgent: Unlike previous synchronous patterns, this agent uses an asynchronous think() method to support concurrent execution. It streams responses using generate_content_stream from Gemini’s asynchronous client.

Roles: There are three roles: Analyst produces concise insights on a specified domain, Critic identifies flaws in each analysis, and Synthesizer merges multiple perspectives into one coherent narrative.

Stage 1 – Parallel Analysts: The run_parallel_analysts() function spawns multiple analysts and runs their think() methods concurrently using asyncio.gather(). The function returns a dictionary mapping analyst names to their outputs.

Stage 2 – Parallel Critics: Each analyst output is critiqued in parallel by a single critic agent. Again, asyncio.gather() is used to run critiques concurrently, producing a dictionary of critiques.

Stage 3 – Synthesis: A synthesizer agent receives all analyses and their corresponding critiques in a single prompt. It returns a unified paragraph summarizing and critiquing the topic.

Main Function: Orchestrates the three stages in sequence: fan‑out (analysts and critics run in parallel) followed by fan‑in (synthesis). The use of nest_asyncio.apply() allows asynchronous code to run within interactive environments.
