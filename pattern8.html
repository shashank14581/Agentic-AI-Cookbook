OVERVIEW

What this pattern teaches

This pattern transitions from synchronous loops to Asynchronous workflows using Python's asyncio. It demonstrates a three-stage pipeline:

1. Fan-Out: Multiple Analyst agents (Economic, Geopolitical, Tech) process a topic concurrently.
2. Validation: A Critic agent scrutinizes each output in parallel.
3. Fan-In: A Synthesizer agent merges all perspectives and critiques into a coherent summary.

CODE

Line-by-line walkthrough

pattern8.py 
import os, asyncio, nest_asyncio
from google import genai
from google.genai import types

class BaseAgent:
    def __init__(self, name, sys_prompt, model="gemini-2.5-flash-lite"):
        self.name = name
        self.sys_prompt = sys_prompt
        self.client = genai.Client(api_key=os.environ["GEMINI_API_KEY"])
        self.model = model

    async def think(self, input_text):
        contents = [types.Content(role="user", parts=[types.Part.from_text(text=input_text)])]
        config = types.GenerateContentConfig(
            system_instruction=[types.Part.from_text(text=self.sys_prompt)]
        )
        response = ""
        stream = await self.client.aio.models.generate_content_stream(
            model=self.model, contents=contents, config=config
        )
        async for chunk in stream:
            if chunk.text: response += chunk.text
        return response

async def run_parallel_analysts(topic):
    analysts = [Analyst("Riya", "economics"), Analyst("Arjun", "risk")]
    async def run(a): return a.name, await a.think(topic)
    return dict(await asyncio.gather(*[run(a) for a in analysts]))

async def run_parallel_critics(outputs):
    async def critique(n, t):
        c = Critic("Dev")
        return n, await c.think(f"{n} said: {t}")
    return dict(await asyncio.gather(*[critique(n, t) for n, t in outputs.items()]))

async def synthesize(analysts, critics):
    s = Synthesizer("Priya")
    data = "\n".join([f"{n}: {t} (Critique: {critics[n]})" for n, t in analysts.items()])
    return await s.think(f"Merge these: {data}")

async def main():
    topic = "The rise of sovereign AI compute"
    analysts = await run_parallel_analysts(topic)
    critiques = await run_parallel_critics(analysts)
    print(await synthesize(analysts, critiques))

if __name__ == "__main__":
    asyncio.run(main())
Mental Model

Lines 6-23: The BaseAgent now uses client.aio (Async IO). This allows the Python script to start one request and move to the next without waiting for the first one to finish.

Lines 25-28: asyncio.gather() is the "Fan-Out" mechanism. It sends multiple analyst prompts to Gemini simultaneously, drastically reducing total execution time.

Lines 30-34: Even the verification step is parallelized. Each analyst's work gets a dedicated critic task running at the same time.

Lines 36-39: The "Fan-In" stage. The Synthesizer acts as the bottleneck where all parallel streams are merged into one intelligent conclusion.

Lines 41-45: The orchestrator manages the sequence: Start parallel analysts → Start parallel critics → Run synthesis.
