Pattern 7: Supervisor Pattern

A supervisor coordinates specialized agents (planner, analyst, copywriter) to complete a task.

Summary

This pattern demonstrates a simple workflow manager. The supervisor keeps track of progress and decides which specialist agent to call next. The state includes a task and optional fields for strategy, insights, and copy. The supervisor’s prompt describes rules for selecting the next step (planner → analyst → copywriter). Each specialized agent generates its part and updates the state. When all fields are filled, the workflow finishes.


import os
import json
from google import genai
from google.genai import types

# BaseAgent defines think() that can optionally enforce JSON schema.
class BaseAgent:
    def __init__(self, name, sys_prompt, model="gemini-2.5-flash-lite"):
        self.name = name
        self.sys_prompt = sys_prompt
        self.client = genai.Client(api_key=os.environ.get("GEMINI_API_KEY"))
        self.model = model
        self.memory = []
    def think(self, input_text, schema=None):
        contents = [types.Content(role="user", parts=[types.Part.from_text(text=input_text)])]
        config = types.GenerateContentConfig(
            system_instruction=[types.Part.from_text(text=self.sys_prompt)],
            thinking_config=types.ThinkingConfig(thinking_budget=0),
            response_mime_type="application/json" if schema else None,
            response_schema=schema
        )
        response = ""
        for chunk in self.client.models.generate_content_stream(model=self.model, contents=contents, config=config):
            if chunk.text:
                print(chunk.text, end="")
                response += chunk.text
        print()
        self.memory.append((input_text, response))
        return response

# Planner, Analyst, Copywriter: specialized workers with run() methods
class Planner(BaseAgent):
    def __init__(self):
        super().__init>("Planner", "You are a marketing strategist. Create a clear strategy.")
    def run(self, state):
        return self.think(json.dumps(state))

class Analyst(BaseAgent):
    def __init__(self):
        super().__init>("Analyst", "You are a data analyst. Generate insights from the context.")
    def run(self, state):
        return self.think(json.dumps(state))

class Copywriter(BaseAgent):
    def __init__(self):
        super().__init>("Copywriter", "You are a marketing copywriter. Write compelling campaign copy.")
    def run(self, state):
        return self.think(json.dumps(state))

# Supervisor: decides next agent based on state
class Supervisor(BaseAgent):
    def __init__(self):
        super().__init>("Supervisor", """
You are a supervisor managing a workflow.
State contains task, strategy, insights, copy.
Rules: if no strategy → planner; if strategy but no insights → analyst; if insights but no copy → copywriter; if all exist → done.
Return JSON: { "next": "planner" | "analyst" | "copywriter" | "done" }
""")
        self.schema = {
            "type": "object",
            "properties": {"next": {"type": "string", "enum": ["planner", "analyst", "copywriter", "done"]}},
            "required": ["next"]
        }
    def route(self, state):
        response = self.think(json.dumps(state), schema=self.schema)
        return json.loads(response)["next"]

# Orchestrate: repeatedly consult supervisor and run specialists until done
def orchestrate():
    state = {"task": "Increase conversion for email campaigns at Big Bazar", "strategy": None, "insights": None, "copy": None}
    planner, analyst, copywriter = Planner(), Analyst(), Copywriter()
    supervisor = Supervisor()
    mapping = {"planner": planner, "analyst": analyst, "copywriter": copywriter}
    while True:
        print("Supervisor deciding...")
        step = supervisor.route(state)
        if step == "done":
            print("Workflow complete")
            break
        agent = mapping[step]
        print(f"{agent.name}: ", end="")
        output = agent.run(state)
        if step == "planner":
            state["strategy"] = output
        elif step == "analyst":
            state["insights"] = output
        elif step == "copywriter":
            state["copy"] = output

# Main entry point
if __name__ == "__main__":
    orchestrate()

BaseAgent: The supervisor pattern reuses BaseAgent from previous patterns. The think() method optionally accepts a JSON schema and configures the Gemini client to return JSON when a schema is provided.

Worker Agents: The Planner, Analyst and Copywriter classes inherit from BaseAgent and define a run() method. Each one takes a serialized state dictionary, prompts the model to generate a strategy, insights, or marketing copy, and returns the generated text.

Supervisor: Extends BaseAgent with a custom prompt that lists the workflow rules. It defines a JSON schema specifying that the model must return a single field "next" with one of four values. The route() method passes the current state through the model and parses the JSON to decide which agent runs next.

Orchestrate: Initializes the state and agent instances. In a loop, it asks the supervisor for the next step. If the supervisor returns "done", the loop exits. Otherwise, it runs the corresponding specialist, updates the state with the result, and repeats. This pattern illustrates how to coordinate multiple agents to complete a structured task.

Main: The script’s entry point simply calls orchestrate() to start the workflow.
